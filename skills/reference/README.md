# Reference Skills

Cross-cutting lookups for known issues, compatibility matrices, and decision aids. These skills are not tied to a single operation — consult them whenever you need to verify compatibility, choose between approaches, or investigate unexpected behavior.

## Sub-Skills

| Skill | Path | Purpose |
|---|---|---|
| Known Issues | `skills/reference/known-issues/` | Documented bugs, workarounds, and status by platform and firmware |
| Compatibility | `skills/reference/compatibility/` | Tool and feature support matrices by platform generation |
| Decision Guides | `skills/reference/decision-guides/` | Structured comparisons to help choose the right tool or approach |

## When to Check

- **Before any firmware update** — check Known Issues for your current and target firmware version
- **When choosing between tools or approaches** — check Decision Guides for structured comparisons
- **When hitting unexplained behavior** — check Known Issues for documented bugs and workarounds
- **Before automating a new platform** — check Compatibility to confirm which features are supported
